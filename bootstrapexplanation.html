Bootstrap Explanations

navbar-brand

I used navbar-brand to be able to fit the logo of the diner into the navbar. This allows me to fit it inside nicely, and let's me adjust the padding, height, etc. Many popular websites do this, as it gives more feeling to the navagation bar.

carousel slide

I used the carousel slide on the site to be able th have it automatically go through pictures. This gives the site a more professional feel, and allows the users to look at something else when visting the site, while making it more active.

carousel-indicators

I also used carousel-indicators to allow the user to see which image was sliding by. This let's them know how many images are on the slide, and if there is any important information that they could be possibly missing out on.

navbar

I used navbar on the website to make it easier for the user to find what they're looking for quickly. Almost every site has a navbar, so creating a useful navbar adds legitimacy to the website for users.

container-fluid

I used container-fluid on the navbar and on the footer, to create a more solid feel to the website. Having a border at the top and bottom of a site helps the user look towards the middle to find the important information that they are looking for.

row

I used row to be able to fit the everything into nice evenly space accurate spaces. It allowed me to create multiple columns, and get everything nice and organized. It made the website's information to be in a neat collective specified area.

sr-only

I used sr-only class to hide information used for screen readers. Screen readers are forms of assistive technology, which help the seeing-impaired and other applications that use such things to say vocally what certain things do. It's to help with compliance for the government (https://www.section508.gov/). Drew Ryan in his youtube video does not explain what it's for, and it was interesting reading about it's purpose.

collapse

I used collapse on a couple of my containers to preserve space to the random user just passing by. Instead of displaying all of the information at once, it allows the user to decide what's important or valuable to them, instead of just having them search through everything that is already presented.

float-*-left

I used float left to help align certain the logo to the left side of the navbar. This way it disrupts the logo from the list items on the navbar and allows the user to feel that the page has more symmetry. The float is a very useful tool and it allows you to divide the page easily and push certain items to one side.

border-white

I used border-white around my navbar to give it a little bit more of design. It helps it pop out more and stand out from the main page of the website as almost an attachment rather that put together. This helps bring creativity to the website and also helps the user differentiate from the information.

Pros of Using Bootstrap

I personally think there is way more pros to using Bootstrap than Wordpress. Bootstrap allows you to literally create you're website, rather than just pick a theme and put what you want on it. With Bootstrap you're able to define how you want your website to look, and how exactly you want it to be spaced out. It allows you to create you own buttons and put images where every you like, and allow you to have the individuality of your website so that it doesn't just resemble a cookie cutter webpage. The navbar functionality and other classes that Bootstrap has makes it an obvious choice in designing your own website.

Cons of Using Bootstrap

The biggest con to using Bootstrap compared to Wordpress is the time it takes to create. With Wordpress you can load a theme and just input the text and images that you need to make your website functional, while with Bootstrap you have to physically code every inch of it. I think once Bootstrap and web design became super easy to a person, I don't think that it would be that much of a hassle though to just create a site using Bootstrap rather than WordPress.
